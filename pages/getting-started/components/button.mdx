### Import

```tsx filename="Exemple.tsx"
import { Button } from "@xefi/x-react/Button";
```

Button Component exporte 2 composants :

- `Button` : Le composant principal pour afficher un bouton.
- `ButtonGroup` : Un composant wrapper pour afficher un groupe de boutons.
